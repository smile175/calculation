# calculate — Windows GUI 計算機（C / Win32）

一個把「計算」做得乾淨俐落的小型桌面應用：純 C、純 Win32 API、無第三方依賴。介面簡潔、反應直接，該有的算術一個也沒少；重點是專注、可靠、好理解。

## 亮點
- 純 Win32 GUI，啟動快、體積小。
- 支援 `+ - * /` 與小數點。
- 可連續輸入表達式（例如 `12.5*3-4/2`）。
- 支援一元負號（例如 `-5+2`）。
- 具備 `AC` 全清、`Del` 單字元刪除。
- 自動移除結果多餘的尾端 0（例如 `2.5000` 會顯示為 `2.5`）。

## 專案結構
- `main.c`：
  - 介面與事件處理（按鈕、顯示區）。
  - 表達式解析與計算（shunting‑yard 變體 + 堆疊）。
- `build.bat`：
  - Windows 上的 GCC 一鍵建置腳本。
- `calculator.exe`：
  - 已編譯好的執行檔。

## 建置方式（Windows）
需求：安裝 MinGW 或其他可用的 GCC。

```bat
build.bat
```

輸出：`calculator.exe`

## 執行方式
直接雙擊 `calculator.exe`，或在命令列輸入：

```bat
calculator.exe
```

## 使用說明
- 使用按鈕輸入數字與運算子。
- `AC`：清空整個表達式。
- `Del`：刪除最後一個字元。
- `=`：計算並顯示結果；上一行會保留原表達式方便確認。

## 計算邏輯（技術簡述）
- 解析採用簡化的 shunting‑yard 流程。
- 兩個堆疊：數值堆疊（double）與運算子堆疊（char）。
- 運算子優先序：`* /` 高於 `+ -`。
- 一元負號支援：當運算子出現在開頭或另一運算子後，視為負號。
- 除以 0 會回傳錯誤並顯示 `Error`。

## 已知限制
- 只支援 `+ - * /` 與小數點，未包含括號或函數。
- 表達式長度上限為 512 字元。
- 未做科學記號輸入（如 `1e-3`）。
